# 🏫 نظام إدارة نقاط المدرسة الابتدائية

## وصف المشروع
موقع ويب عربي متكامل لإدارة نقاط تلاميذ المدرسة الابتدائية، مبني بـ React + Supabase.

---

## 🚀 خطوات الإعداد

### 1. إنشاء جداول Supabase
افتح مشروعك في Supabase → SQL Editor → انسخ والصق محتوى ملف `SUPABASE_SCHEMA.sql` واضغط Run.

### 2. ضبط متغيرات البيئة
افتح ملف `.env` وأضف:
```
REACT_APP_SUPABASE_URL=https://xxxx.supabase.co
REACT_APP_SUPABASE_ANON_KEY=eyJhbG...
```
> يمكنك إيجاد هذه القيم في: Supabase Dashboard → Settings → API

### 3. تشغيل المشروع
```bash
npm install
npm start
```

## 🗂️ هيكل الوحدات الدراسية

| الوحدة | المواد | الملاحظة |
|--------|--------|----------|
| عربية | تواصل / إنتاج كتابي / قراءة / قواعد اللغة | جميع السنوات |
| علمية | رياضيات / إيقاظ علمي / تربية تكنولوجية | جميع السنوات |
| اجتماعية | جغرافيا / تاريخ / مدنية / اسلامية | السنوات 5 و 6 = كل المواد |
| اجتماعية | اسلامية فقط | السنوات 1 إلى 4 |
| فنية | تشكيلية / موسيقية / بدنية | جميع السنوات |
| فرنسية | expression écrite / expression orale | جميع السنوات |
| Anglais | Anglais | السنوات 4، 5 و 6 فقط |

---

## ✨ المميزات

- **الوحدات**: عرض جميع الوحدات في بطاقات قابلة للسحب وإعادة الترتيب
- **الترتيب**: تصنيف التلاميذ تلقائياً حسب المعدل العام مع معدل كل وحدة
- **النقاط**: جدول نقاط تفاعلي مع حفظ تلقائي عند الخروج من الخانة + معدل كل وحدة
- **تعديل الاسم**: زر ✏️ بجانب كل تلميذ لتعديل اسمه مباشرة
- **دعم RTL كامل**: واجهة عربية بالكامل
